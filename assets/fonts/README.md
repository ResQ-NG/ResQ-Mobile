# Metropolis font

Upright weights are loaded in `src/theme/fonts.ts` (all .otf):

- Thin, ExtraLight, Light, Regular, Medium, SemiBold, Bold, ExtraBold, Black

**Tailwind classes:** `font-metropolis-thin`, `font-metropolis-extralight`, `font-metropolis-light`, `font-metropolis-regular`, `font-metropolis-medium`, `font-metropolis-semibold`, `font-metropolis-bold`, `font-metropolis-extrabold`, `font-metropolis-black`. Default `font-sans` uses Regular.

Italic variants (e.g. `Metropolis-LightItalic.otf`) are in this folder; add them to `src/theme/fonts.ts` and `tailwind.config.js` if you need them.
